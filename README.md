# PauperWave

Blog & League Manager for the PauperWave association. 
Built with [Nuxt 4](https://nuxt.com) and Nitro for server-side rendering and static site generation.

## Setup

Make sure to install dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
